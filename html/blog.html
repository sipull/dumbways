Diesnatalis Fakultas Teknik

Duration

12 Januari 2023 - 11 Februari 2021
1 month

Technologies

React Js
JavaScript
Node Js
Socket IO

Setelah sekitar sembilan tahun diresmikan, Fakultas Teknik (FT) Universitas Hasyim Asy’ari (Unhasy), baru kali ini Badan Eksekutif Mahasiswa (BEM) merayakan Dies Natalis dengan acara meriah di halaman gedung C Unhasy, Senin (18/7/2022). “Kegiatan ini merupakan serangkaian program dari BEM Fakultas Teknik dengan harapan menjadikan acara ini sebagai titik balik semangat baru bagi mahasiswa Fakultas Teknik setelah pandemi,” ungkap ketua pelaksana, Nia. Pada malam puncak ini, selain potong tumpeng ada pembagian hadiah bagi pemenang lomba yang diadakan sejak Jumat (8/7/2022) lalu. Lomba tersebut berupa Futsal dan Badminton untuk tingkat Unhasy. Kompetisi game Mobile Legends untuk regional Jawa Timur. Selain itu, penampilan group band Rain in July musik Jombang turut memeriahkan di malam puncak. “Fakultas Teknik baru kali ini mengadakan acara Dies Natalis. Untuk itu, harapan saya semoga ini menjadi program kegiatan rutin untuk ke depannya dan semoga bisa lebih bagus lagi untuk meningkatkan Dies Natalis Fakultas Teknik,” ungkap salah satu penonton, Mabruri.
